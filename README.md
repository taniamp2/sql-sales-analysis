# SQL Sales Performance Analysis

This project analyzes transactional sales data using PostgreSQL.

## Objectives
- Calculate total revenue
- Analyze revenue by region
- Identify top-performing products
- Evaluate category contribution to total sales

## Key Insights

- Total revenue: 4780
- North America generated the highest revenue (2500)
- Laptops generated the highest revenue despite low unit sales
- Electronics account for the majority of total revenue

## Tools Used
- PostgreSQL
- SQL
- VS Code
SQL project analyzing sales data to extract key business insights such as revenue by region, product performance, and category contribution.
